# 🚀 Tic Tac Toe Game   

A clean and modular implementation of the classic **Tic Tac Toe** game built using modern programming practices.

---

## 📌 Features

- 🎮 Two-player gameplay  
- 🧠 Win/draw detection logic  
- 🔄 Game restart functionality  
- ⚡ Optimized game logic  
- 🧱 Modular code structure
